# @sentra/ui

React UI primitives for Sentra AI products. These components are intentionally
small, auditable, and token-driven.

## Import

```tsx
import {
  SentraLogo,
  SentraMark,
  SentraButton,
  SentraCard,
  SentraBadge,
  SentraPanel,
} from '@sentra/ui'
import '@sentra/design-token/css'
import '@sentra/ui/styles'
```

## Components

- `SentraMark` — inline SVG icon mark.
- `SentraLogo` — horizontal brand lockup.
- `SentraButton` — CTA / secondary / ghost button.
- `SentraCard` — dark or light card primitive.
- `SentraBadge` — status / brand badge.
- `SentraPanel` — structured product panel.
