# @sentra/design-token

Source of truth for Sentra Artificial Intelligence 2026 brand tokens.

## Exports

- `@sentra/design-token` — typed token objects.
- `@sentra/design-token/tailwind` — Tailwind preset.
- `@sentra/design-token/css` — CSS variables.
- `@sentra/design-token/assets/*` — SVG/PDF brand assets.

## Usage

```ts
import {
  sentraColors,
  sentraTypography,
  sentraLogo,
} from '@sentra/design-token'
```

```ts
import sentraTailwindPreset from '@sentra/design-token/tailwind'
```

```ts
import '@sentra/design-token/css'
```
