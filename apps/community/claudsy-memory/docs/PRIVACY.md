# Privacy Policy

## Data Collection

Claudsy Memory collects and stores facts extracted from AI agent conversations for memory persistence.

## Data Storage

- All data stored locally on user's machine
- No data transmitted to external servers
- SQLite database with encryption optional

## Data Usage

- Facts used to provide memory continuity for AI agents
- No personal data collection
- No analytics or tracking

## Data Retention

- Facts retained until manually deleted or naturally forgotten via decay
- Session logs archived after 14 days
- Users can delete all data at any time

## Data Sharing

- No data sharing with third parties
- Local-only operation
- No cloud backups by default

## User Rights

- Right to access stored facts
- Right to delete all data
- Right to export data

## Security

- Local storage prevents unauthorized access
- No network exposure
- File permissions control access

## Contact

For privacy concerns, contact privacy@claudsy.com
