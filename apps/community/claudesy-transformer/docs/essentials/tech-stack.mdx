We use a modern, high-performance tech stack to ensure the best possible experience for our users and developers.

## Frontend
- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS v3.4](https://tailwindcss.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/) & [shadcn/ui](https://ui.shadcn.com/)

## Backend & Data
- **Runtime**: Node.js 20
- **Database**: [PostgreSQL (Supabase)](https://supabase.com/)
- **ORM**: [Prisma](https://www.prisma.io/)
- **Validation**: [Zod](https://zod.dev/)

## AI Providers
- **Anthropic SDK**: v0.39
- **Google Generative AI**: v0.24
- **Mistral SDK**: v1.5
- **OpenAI SDK**: v4.85

## Tooling
- **CI/CD**: GitHub Actions
- **Testing**: [Vitest](https://vitest.dev/)
- **Package Manager**: [pnpm v9](https://pnpm.io/)
