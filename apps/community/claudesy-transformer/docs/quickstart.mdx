## Prerequisites

Before setting up Prompt Transformer, ensure you have the following installed:
- [Node.js 20+](https://nodejs.org/)
- [pnpm 9+](https://pnpm.io/)
- [Docker](https://www.docker.com/) (optional, for local Prisma database)

## Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/claudesy/theabyss.git
cd theabyss/app/ctev2
```

### 2. Install dependencies
```bash
pnpm install
```

### 3. Environment Variables
Copy the example environment file and fill in your API keys:
```bash
cp .env.example .env
```

### 4. Database Setup
Generate and migrate the Prisma schema:
```bash
npm run db:generate
npm run db:migrate
```

If your target database already contains the pre-Prisma baseline schema, use the migration deployment flow instead:

```powershell
./scripts/apply-prisma-migrations.ps1 -MarkInitApplied
```

See [Database Migrations](/database-migrations) for the full adoption workflow.

### 5. Start Development Server
```bash
pnpm dev
```
Open [http://localhost:3003](http://localhost:3003) to see your app in action.

## Troubleshooting

If you encounter issues with database connections, ensure your Supabase or local postgres instance is running correctly.

<Tip>
  Use `npm run test` to verify your installation with our pre-built Vitest suite.
</Tip>
