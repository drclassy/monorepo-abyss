# Community Domain Steering

**Domain:** Open Innovation and R&D Tools
**Compliance Level:** STANDARD
**Revision:** 2026.04.06

## Domain Overview

The Community domain fosters innovation through specialized AI tools, personal medical websites, and experimental memory engines.

### Applications
- `claudsy-memory`: Long-term AI memory persistence engine.
- `claudesy-transformer`: Prompt engineering optimization platform.
- `daf-website`: Specialized medical practice digital presence.

## Innovation Standards

### Technical Excellence
- `claudsy-memory`: Uses Ebbinghaus Decay logic for intelligent information retention.
- `claudesy-transformer`: Focuses on deterministic LLM output through advanced prompt refinement.

### Aesthetic Consistency
- Even personal websites (e.g., `daf-website`) must adhere to premium design standards (Cream, Charcoal, Gold palette).

## Technical Standards

### Hybrid Architecture
- Engine components may utilize Python (FastAPI/Core) where appropriate.
- User Interfaces must use Next.js 15+ and React 19.
