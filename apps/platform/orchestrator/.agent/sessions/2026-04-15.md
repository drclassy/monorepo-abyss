## 📝 Technical Decisions

1. **StepLog interface**: Menggunakan string untuk dates (ISO format) untuk
   kompatibilitas dengan JSON Prisma
2. **Status enum**: Menggunakan string literals + type assertion untuk
   menghindari runtime import issues
3. **Execution context**: setExecutionContext() method dipanggil sebelum
   xecute() untuk setup repository
4. **Error handling**: Try-catch di FlowsService untuk mark execution as FAILED
   jika saga gagal
