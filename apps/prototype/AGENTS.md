# Prototype Domain Steering

**Domain:** Incubator and Research Sandbox
**Compliance Level:** MINIMAL
**Revision:** 2026.04.06

## Domain Overview

The Prototype domain is a sandbox environment for rapid experimentation, emerging technology research, and Small Language Model (SLM) testing.

## Operational Rules

### Flexibility
- Relaxed test coverage requirements during initial POC phases.
- Direct implementation of experimental external packages is allowed.

### Graduation Path
- Successful prototypes must be documented in a Decision Record (ADR) before migrating to a production domain (e.g., Healthcare or Academic).

## Safety Constraints
- Basic security protocols (Authentication/Encryption) still apply.
- No PII/PHI usage allowed in the sandbox environment.
