# Clinical Reference Skill

## When to Use
- Chief menanyakan dosis obat untuk pasien
- Chief menanyakan interaksi obat (DDI)
- Chief menanyakan kontraindikasi
- Chief menanyakan pedoman pengobatan klinis
- Chief membutuhkan informasi farmakologi cepat

## Procedure
1. Identifikasi obat yang ditanyakan
2. Berikan informasi: dosis standar, rute pemberian, frekuensi, durasi
3. Jika ada interaksi obat, sebutkan mekanisme dan tingkat keparahan
4. Jika ada kontraindikasi, sebutkan secara eksplisit
5. Selalu sertakan disclaimer: informasi ini untuk referensi, bukan pengganti keputusan klinis
6. JikaChief menanyakan regimen khusus (mis. pediatrik, geriatrik, gangguan ginjal), sesuaikan

## Pitfalls
- Jangan memberikan dosis absolut tanpa konteks (berat badan, usia, fungsi ginjal)
- Jangan menggantikan keputusan dokter yang merawat
- Selalu verifikasi dengan sumber terkini
- Data obat yang tidak familiar → akui dan sarankan Chief merujuk ke referensi resmi
- Jangan memberikan rekomendasi off-label tanpa peringatan eksplisit

## Verification
- Verifikasi dosis dengan pedoman resmi (ISO, FDA, Kemenkes)
- Cross-check interaksi obat dengan database DDI resmi
- Pastikan rekomendasi sesuai konteks klinis Chief
