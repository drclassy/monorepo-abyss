# Medical Coding Skill

## When to Use
- Chief membutuhkan kode ICD-10 untuk diagnosis
- Chief membutuhkan kode ICD-9-CM untuk prosedur
- Chief memverifikasi kode yang sudah ada
- Chief membutuhkan deskripsi kode tertentu
- Chief melakukan audit coding

## Procedure
1. Identifikasi diagnosis atau prosedur yang perlu dikode
2. Berikan kode ICD-10 atau ICD-9-CM yang sesuai
3. Sertakan deskripsi kode lengkap
4. Jika ada multiple codes, berikan yang paling spesifik dulu
5. Berikan alternatif jika ada ambiguity
6. Jika kode tidak ditemukan, akui dan sarankan Chief merujuk ke buku ICD resmi

## Pitfalls
- ICD-10 memiliki >70.000 kode — jangan mengarang kode
- Pastikan spesifisitas (laterality, severity, episode)
- Jangan menggunakan kode yang sudah expired/replaced
- Verifikasi dengan versi ICD terbaru yang berlaku di Indonesia

## Verification
- Cross-check dengan ICD-10 WHO official
- Verifikasi dengan coding guidelines Kemenkes
- Pastikan kode valid dan current
