3 Pendekatan untuk Agent UNICOM di Cursor IDE

Semua extension berjalan dalam satu IDE — bagaimana mereka bisa saling berkomunikasi?

A — UNICOM sebagai MCP Server ⭐ Recommended

Model Context Protocol — bahasa native Claude Code & Roocode

Termudah · Paling Native
// .mcp.json (sudah ada di repo!)
Claude Code ext
↕ MCP client
Roocode ext

✅ Keunggulan
  • Claude Code & Roocode sudah support MCP native
  • .mcp.json sudah ada di repo
  • No new dependency — pakai JSON-RPC 2.0
  • Human-in-the-loop by design
⚠️ Batasan
  • Codex belum pasti support MCP
  • Perlu build custom MCP server (tapi kecil)
  • MCP default = request/response, bukan push
