## 1. Mission Objective

Build a standalone local HTML tool named:

```txt
sentra-context-engine.html
```

Target location:

```txt
D:\Devop\clausytorium\Genesis Framework\sentra-context-engine.html
```

This tool helps Chief generate structured AI context capsules for coding agents such as:

- Claude Code
- Cursor
- Taskmaster AI

The tool must allow Chief to:

1. Open the HTML file directly in Chrome using `file://`.
2. Select relevant context blocks.
3. Select exactly one agent role.
4. Compose a clean context capsule.
5. Copy the full context into clipboard with one click.
6. Refresh the file after Claude Code updates project context.

Correct mental model:

```txt
Sentra Context Capsule = briefing pack for AI coding agents
```

This is not MCP.\
This is not RAG.\
This is not a server application.\
This is not an agent runtime.\
This is a local, reliable, browser-based context composition tool.

### Copy Button

Behavior:

```txt
Click "Copy All" copies the full composer content.
After success, button text becomes "✓ Copied!" for 2 seconds.
```

Clipboard implementation:

1. First try:

```js
navigator.clipboard.writeText(text)
```

2. Fallback:

```js
textarea.select()
document.execCommand("copy")
```

3. If both fail, show visible message:

```txt
Clipboard blocked. Please manually select and copy from composer.
```
