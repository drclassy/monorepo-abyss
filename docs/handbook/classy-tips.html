01 Claude Code Power Moves

1 Gunakan @codebase di awal prompt

Saat mulai sesi baru, beri @codebase terlebih dahulu agar Claude Code memuat seluruh konteks repo. Setelah konteks cukup, gunakan @filename untuk fokus ke file spesifik.

Efisiensi token Context loading

2 Shift+Tab untuk mode cycling

Tekan Shift+Tab berulang kali untuk berganti antara Normal → Auto → Plan mode. Di plan mode, ketik GO di baris tersendiri untuk eksekusi.

Keyboard shortcut

3 Double-Esc untuk undo cepat

Double-Esc (Esc Esc) melakukan rewind perubahan terakhir. Berguna saat Claude membuat edit yang tidak diinginkan — jauh lebih cepat dari /rewind.

Keyboard shortcut

4 /status sebelum setiap sesi panjang

Cek penggunaan token dan model aktif dengan /status. Jika konteks sudah di atas 60%, lakukan /compact focus on [area] sebelum melanjutkan.

Opsional
02 Healthcare Coding Workflow

5 PHI selalu dikode ulang di test

Jangan pernah gunakan data riil pasien (NIK, BPJS, nama lengkap) di test fixture. Gunakan generator data sintetis: nik: '000' + randomDigits(13).

Keamanan

6 Jalankan /sentra-review sebelum push

Perintah /sentra-review menjalankan code review otomatis untuk healthcare apps: cek @Exclude(), Swagger @ApiOperation, dan arsitektur NestJS.

Quality gate

7 /phi-check di setiap commit healthcare

Sebelum git add, jalankan /phi-check untuk memastikan tidak ada PHI/PII yang bocor ke staged changes.

Keamanan

8 Class C tasks → branch terpisah

Untuk perubahan database, infra, atau PHI: selalu buat branch baru git checkout -b agent/task-$(date +%s). Jangan kerjakan langsung di main/develop.

Best practice
03 Monorepo Navigation

9 cd shortcuts untuk apps

Buat alias di ~/.bashrc / ~/.zshrc:
alias cda='cd /v/sentra-artificial-intelligence/abyss-monorepo/apps'
alias cdp='cd /v/sentra-artificial-intelligence/abyss-monorepo/packages'

Dev speed

10 pnpm filter untuk build cepat

Gunakan --filter untuk menargetkan proyek spesifik:
pnpm --filter @the-abyss/sentra-assist dev
pnpm --filter @the-abyss/intelligenceboard build

Build

11 Turbo cache untuk CI lokal

Jalankan pnpm build --filter=[HEAD~1] untuk build only-affected packages, mirip CI pipeline.

CI/CD

12 .agent/ files sebagai cheat sheet

Buka .agent/CONTEXT.md untuk arsitektur, .agent/HANDOFF.md untuk progress terkini, dan .agent/LESSONS.md untuk belajar dari kesalahan.

Onboarding
04 Hidden Gems

13 Cursor AI + Claude Code combo

Gunakan keduanya untuk hal berbeda. Cursor AI (Ctrl+L) untuk inline completion dan single-file edits. Claude Code untuk multi-file agentic work, planning, test runs, dan git operations. Jangan duplikasi effort.

14 Prompt snippets di .cursor/snippets/

Buat snippet untuk prompt berulang: .cursor/snippets/healthcare-review.md, .cursor/snippets/api-design.md. Akses via @snippet.

15 claude --list-sessions

Untuk resume sesi lama tanpa scrolling history: claude --resume <session-id>. ID bisa didapat dari claude --list-sessions.

16 Export diff sebagai PR description

git diff --stat HEAD > pr-summary.txt lalu paste ke PR description. Tambah --name-only untuk daftar file yang berubah.

17 Auto-compact di tengah task

Jika context mencapai ~70%, lakukan /compact focus on [area] secara manual. Jangan tunggu auto-compact — bisa terjadi di momen tidak tepat.

18 Git stash sebelum eksperimen

Sebelum mencoba pendekatan baru: git stash push -u -m "pre-experiment". Kembali dengan git stash pop atau drop dengan git stash drop.
