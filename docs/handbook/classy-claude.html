01 · Core Workflow

Urutan default eksekusi Classy — selalu mulai dari sini.
SkillFungsiKategori
$classy-using Reset ke baseline Classy — pilih mode yang tepat sebelum mulai, atau saat eksekusi mulai menyimpang Core
$classy-brainstorming Frame masalah sebelum menyelesaikannya — clarify tujuan nyata, surface constraints, timbang 2-3 pendekatan Core
$classy-plan Ubah arah yang sudah disepakati menjadi rencana implementasi terurut, executable, dengan rollback Core
$classy-implement Eksekusi plan dalam delta kecil yang reviewable — verifikasi dekat setiap perubahan Core
$classy-verify Hasilkan bukti konkret bahwa perubahan bekerja — test, build, typecheck, lint, smoke run Core
$classy-finish Tutup task secara jujur — state readiness, list cleanup, nama handoff items, berikan sinyal READY Core
