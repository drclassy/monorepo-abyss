🤖 Agent Modes
Agent Full autonomy — edits files, runs terminal, searches web. Use for implementing features end-to-end.
Ask Read-only exploration. Answers questions about your codebase without modifying anything.
Plan Researches + asks clarifying questions → generates a step-by-step plan → waits for your approval before coding. Best for complex multi-file changes.
🎯 Rule of thumb: Ask → understand, Plan → design, Agent → execute.

📋 Rules
Location: .cursor/rules/ (project) · Settings → Rules (global)
Always Injected into every agent context, every time. Use for non-negotiable conventions.
Auto Attached Included when files match a glob (e.g. **/*.ts). Set via globs frontmatter.
Agent Requested Agent decides whether to pull it in. Requires a description field.
Manual Only included when you type @ruleName explicitly.
Rule File Anatomy (MDC format)
--- description: "TypeScript conventions" globs: ["**/*.ts", "**/*.tsx"] alwaysApply: false --- Use strict TypeScript. No `any`. Prefer `unknown`. Use Zod for all external input validation. ES modules only — never CommonJS.
💡 Rules = persistent memory. Put team conventions here so you never explain them again.

⚡ Power Tips
🧠 Plan before big changes. Shift+Tab or /plan → Agent researches → you approve → then it codes. Prevents wrong-direction sprints.
📎 @Codebase for unknown locations. Better than @Files when you don't know exactly which file has what you need — semantic search finds it.
📋 Rules = persistent conventions. Put team patterns in Always rules. Agent follows them every session without re-explaining.
🔄 Checkpoints = safe undo. Ctrl+Shift+J jumps between chat checkpoints. Rollback if Agent went sideways.
🗜️ Long sessions getting slow? Type /compress → summarizes conversation, frees context window without losing key info.
⚡ Ctrl+K for small edits. Don't open full Agent for single-function changes. Inline Edit is faster and stays in place.
🌐 @Link for live docs. Type @https://docs.example.com/api — Agent fetches and reads the page as context.
🔒 Privacy mode available. Settings → disable telemetry + keep code indexing local only. Zero code sent to servers.
🔁 Auto-run for trusted tasks. /auto-run on lets Agent run terminal commands without asking each time. Toggle off for sensitive repos.
