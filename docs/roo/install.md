# Install Guide

## Step 1 — Copy files

Copy these into the root of `abyss-monorepo`:

```text
.roomodes
.roo/
```

Optional docs:

```text
docs/roo/
```

## Step 2 — Open IDE

Open the ABYSS monorepo root in Cursor or VS Code.

## Step 3 — Open Roo/ZooCode

Open the extension panel and choose project modes.

## Step 4 — Assign models

Manually assign preferred model per mode. The extension remembers the last model used per mode through sticky model behavior.

## Step 5 — Test safely

Use `Sentra Search` first. Do not start with `Sentra Deep`.

Recommended first command:

```text
Use Sentra Search mode. Analyze package cleanup readiness for packages/shared only. Do not edit files. Return paths, references, and risks.
```
