# Archive index

This folder stores documentation that is kept for history, auditability, or
reference, but is not part of the primary active doc surface anymore.

## Archived folders

| Folder | Reason |
| --- | --- |
| `docs/archive/cursor-handbook/` | Legacy Cursor handbook packaging subproject and its supporting assets. |
| `docs/archive/handbook/` | Static HTML handbook exports and local font assets. |
| `docs/archive/plans/` | Historical one-off execution plans no longer treated as active root docs. |
| `docs/archive/research/` | Old workflow-state notes and assistant operating scratchpads. |
| `docs/archive/task/` | Historical task handoff bundles and RAG work-item documents. |

## Archive rules

- Keep files readable and linkable, but do not treat them as the primary source
  of truth.
- Prefer editing active docs instead of reviving archived copies.
- If an archived document becomes active again, move it back to the proper
  active folder and update links deliberately.
